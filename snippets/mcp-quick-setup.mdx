## Quick MCP Setup

### 1. Install MCP Server
```bash
npx mint-mcp add docs.probe.dev
```

### 2. Set API Key
```bash
export PROBE_API_TOKEN="your_api_key_here"
```

### 3. Configure Your Client

<CodeGroup>
```json Cursor (~/.cursor/mcp.json)
{
  "mcpServers": {
    "probe-dev": {
      "command": "npx",
      "args": ["mint-mcp", "run", "docs.probe.dev"],
      "env": {
        "PROBE_API_TOKEN": "your_actual_api_key_here"
      }
    }
  }
}
```

```json Claude Desktop
{
  "mcpServers": {
    "probe-dev": {
      "command": "npx",
      "args": ["mint-mcp", "run", "docs.probe.dev"],
      "env": {
        "PROBE_API_TOKEN": "${PROBE_API_TOKEN}"
      }
    }
  }
}
```
</CodeGroup>

### 4. Test
Ask your AI: *"Search the Probe.dev docs for authentication"*
