## Quick MCP Setup

### 1. Get Your API Key
Get your API token from [probe.dev/dashboard](https://www.probe.dev/dashboard)

### 2. Configure Your Client

<CodeGroup>
```json Cursor (~/.cursor/mcp.json)
{
  "mcpServers": {
    "probe-dev": {
      "url": "https://mcp.probe.dev/",
      "headers": {
        "Authorization": "Bearer your-api-token-here"
      }
    }
  }
}
```

```json Claude Desktop
{
  "mcpServers": {
    "probe-dev": {
      "url": "https://mcp.probe.dev/",
      "headers": {
        "Authorization": "Bearer your-api-token-here"
      }
    }
  }
}
```
</CodeGroup>

### 3. Test
Ask your AI: *"Search the Probe.dev docs for authentication"*
