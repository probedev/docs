## Professional Codec Variants

### Apple ProRes Family
- **ProRes 422 Proxy** - Low bandwidth editing
- **ProRes 422 LT** - Standard quality editing  
- **ProRes 422** - High quality standard
- **ProRes 422 HQ** - Higher quality for finishing
- **ProRes 4444** - Full resolution with alpha
- **ProRes 4444 XQ** - Highest quality variant

### DV (Digital Video) Variants
- **DV25** - Standard DV (25 Mbps)
- **DVCPRO25** - Professional DV variant
- **DVCPRO50** - Higher quality DV (50 Mbps)
- **DVCPRO100** - Broadcast quality DV
- **DVCPRO HD** - High-definition DV

### H.264 Subtypes
- **Baseline** - Basic compatibility
- **Main** - Standard quality/features
- **High** - Advanced features, best compression
- **High 10** - 10-bit color depth
- **High 4:2:2** - Professional chroma sampling

## Codec Analysis Examples

### Identify Codec and Profile
```bash
curl -G https://api.probe.dev/v1/probe/file \
  --data-urlencode "token=${PROBE_API_TOKEN}" \
  --data-urlencode "only=probe_report" \
  --data-urlencode "url=https://example.com/video.mp4"
```

### Detailed Codec Information
```bash
curl -G https://api.probe.dev/v1/probe/file \
  --data-urlencode "token=${PROBE_API_TOKEN}" \
  --data-urlencode "only=ffprobe" \
  --data-urlencode "ffprobe[show_streams]=true" \
  --data-urlencode "url=https://example.com/video.mov"
```

## Codec Selection Guide

### For Web Distribution
```
H.264 High Profile - Best compatibility
H.265 Main Profile - Better compression, newer devices
AV1 - Best compression, cutting-edge browsers
VP9 - Good compression, open source
```

### For Professional Workflows
```
ProRes 422 HQ - Apple ecosystems, high quality
DNxHD/DNxHR - Avid workflows, broadcast
H.264 High Profile - Universal compatibility
```

### For Archival Storage
```
FFV1 - Lossless, open standard
ProRes 4444 - High quality with alpha
H.264 Lossless - Good compression, wide support
```

## Performance Characteristics

### Encoding Complexity
| **Codec** | **Encoding Speed** | **Quality** | **File Size** | **Compatibility** |
|-----------|-------------------|-------------|---------------|-------------------|
| **H.264** | Fast | Good | Medium | Excellent |
| **H.265** | Slow | Excellent | Small | Good |
| **AV1** | Very Slow | Excellent | Very Small | Limited |
| **VP9** | Slow | Very Good | Small | Good |
| **ProRes** | Very Fast | Excellent | Large | Apple-focused |
